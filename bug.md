# Bug记录
## 1. 四元数偏差计算
- 日期：2026-01-20
- 具体描述：四元数偏差经过标量符号(Relay(·))相乘后，会陷入$[-1,0,0]\rightarrow[1,0,0]$的死循环，输出的四元数呈现正弦周期的波形。
- 原因分析：四元数偏差计算的符号反了，但不知道为什么
- 处置措施：把Relay的参数
```json
{
    open:1,
    close:-1,
},
```
改为：
```json
{
    open:-1,
    close:1,
},
```
## 2. 太阳入射角和光电功率计算
- 日期：2026-01-22
- 具体描述：SOC持续下降，不再回升，一直达到并保持0。
- 原因分析：如果姿态稳定时，太阳板转到背光面，充电为0，SOC就会持续下降。
- 处置措施：修改最终姿态稳定的角度。
    - 对于当太阳矢量和太阳板法向量为\[1,0,0\]时，需保证$2(q_2^2+q_3^2)<1$。
## 3. DC-DC变换器占空比
- 日期：2026-01-27
- 具体描述：选择Buck变换器，太阳板电压$V_{pv}=V_{bus}\cdot D$，运行后占空比会一直处于上限限幅。
- 原因分析：占空比对功率变化太敏感，可能的精度误差也会被识别为变化。
- 处置措施：添加阈值过滤，变化超过阈值才会改动占空比。

## 4. 重力梯度力矩数量级在$10^{-28}$
- 日期：2026-02-04
- 具体描述：重力梯度力矩太小，应该在$10^{-6}$~$10^{-5}$的级别。
- 原因分析：卫星转动惯量设置较小；同时$r_i$向量拼接4维向量时，常数项位置放错了。
- 处置措施：调整常数项位置，数量级恢复到$10^{-10}$。

## 5. 四元数偏差公式
- 日期：2026-03-04
- 具体描述：求四元数偏差的公式，写错了。应该是$q_e=q_d\otimes q_a^{*}$
- 原因分析：根据四元数的结合律重新推算得到
- 处置措施：对实际四元数求共轭
